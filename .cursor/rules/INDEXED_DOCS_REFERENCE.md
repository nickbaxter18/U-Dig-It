## Usage Tips

1. **Use @ mentions in questions**: "How do I create an RLS policy? @Supabase RLS"
2. **Multiple docs**: "@Supabase Auth @https://supabase.com/docs/guides/security how do I secure auth?"
3. **Auto-attached**: Rules will prompt me to use these automatically
4. **Explore**: Type `@` to see all available docs

## Maintenance

### Re-indexing Schedule
- **Weekly**: Core infrastructure (Supabase, Next.js)
- **Monthly**: UI libraries, testing tools
- **Quarterly**: Integrations, starter templates

### Adding New Docs
1. Go to Settings → Indexing & Docs
2. Click "+ Add Doc"
3. Enter URL or select from library
4. Update this reference file
5. Update relevant auto-reference rule
